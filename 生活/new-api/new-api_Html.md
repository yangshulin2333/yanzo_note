## üöÄ Quick Start

[](https://github.com/QuantumNous/new-api#-quick-start)

### Using Docker Compose (Recommended)

[](https://github.com/QuantumNous/new-api#using-docker-compose-recommended)

```shell
# Clone the project
git clone https://github.com/QuantumNous/new-api.git
cd new-api

# Edit docker-compose.yml configuration
nano docker-compose.yml

# Start the service
docker-compose up -d
```

**Using Docker Commands**

```shell
# Pull the latest image
docker pull calciumion/new-api:latest

# Using SQLite (default)
docker run --name new-api -d --restart always \
  -p 3000:3000 \
  -e TZ=Asia/Shanghai \
  -v ./data:/data \
  calciumion/new-api:latest

# Using MySQL
docker run --name new-api -d --restart always \
  -p 3000:3000 \
  -e SQL_DSN="root:123456@tcp(localhost:3306)/oneapi" \
  -e TZ=Asia/Shanghai \
  -v ./data:/data \
  calciumion/new-api:latest
```

> **üí° Tip:**¬†`-v ./data:/data`¬†will save data in the¬†`data`¬†folder of the current directory, you can also change it to an absolute path like¬†`-v /your/custom/path:/data`

üéâ After deployment is complete, visit¬†`http://localhost:3000`¬†to start using!
