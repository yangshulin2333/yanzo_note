# 开发环境说明文档

此目录旨在提供一个清晰、易维护的开发环境。

## 结构概览
- **IDEs**: 集成开发环境安装目录。
- **Runtimes**: 各语言运行时环境（JDK, Python, NodeJS等）。
- **Tools**: 开发辅助工具（Maven, Git, Docker等）。
- **Servers**: 本地服务器软件。
- **Configs**: 配置文件备份。
- **Repository**: 本地依赖仓库。
- **Scripts**: 自动化脚本。
- **Docs**: 环境相关文档。

## 环境变量
请参考 `Scripts/set-env.bat` (Windows) 或 `Scripts/set-env.sh` (Linux/macOS) 配置环境变量。

## 配置文件版本控制
建议将 `Configs` 文件夹初始化为 Git 仓库进行版本控制。
