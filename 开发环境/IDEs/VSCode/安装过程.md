
### 开发环境配置流程

为了让你能完全掌控，我们一步步来：

#### 1. 下载与安装

- 去官网下载并安装 [VS Code](https://code.visualstudio.com/)。
    
- 安装时记得勾选 **"Add to PATH"**（添加到环境变量）。
    

#### 2. 安装 Python 核心插件

- 打开 VS Code，点击左侧栏最下面的四个方块图标（Extensions/插件）。
    
- 搜索框输入 `Python`，选择由 **Microsoft** 发布的那一个，点击 **Install**。
    

#### 3. 配置 Python 解释器

- 按快捷键 `Ctrl + Shift + P`，输入 `Python: Select Interpreter`。
    
- 选择你刚才看到的 **Python 3.14.2**。
    

---

### 如何开始我们的第一个实战脚本？

1. 在电脑上新建一个文件夹，比如叫 `GooseMind_Py`。
    
2. 在 VS Code 中点击 `File -> Open Folder`，选中这个文件夹。
    
3. 点击左侧的新建文件图标，创建一个文件叫 `screen_test.py`。
    
4. 将我刚才给你的那段代码粘贴进去。

### 解决“不可抗力”：安装依赖库

在 VS Code 界面最下方，你会看到一个 **Terminal（终端）** 窗口。如果没有，按 `Ctrl +` ` (键盘 Tab 键上方那个键)。

在终端里依次输入并回车：

### 设置中文
- **打开扩展商店**： 点击 VS Code 左侧侧边栏的 **方块图标**（Extensions），或者直接按下快捷键 `Ctrl` + `Shift` + `X`。
    
- **搜索中文插件**： 在搜索框中输入 `Chinese`。
    
- **安装插件**： 找到列表中的 **Chinese (Simplified) (简体中文)** 插件（通常是第一个，图标是个地球），点击蓝色的 **Install** 按钮。
    
- **重启生效**： 安装完成后，右下角通常会弹出一个提示框。点击 **Change Language and Restart**（更改语言并重启）。
