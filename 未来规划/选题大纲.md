# 游戏开发职业规划：选题大纲

## 一、技术深度
- ### 引擎专精
  - 深入学习 Unity 或 Unreal Engine 的一个，并成为该领域的专家。
  - 研究引擎底层架构和渲染管线。
- ### 图形学
  - 学习高级着色器（Shader）编程。
  - 探索光线追踪、全局光照等前沿技术。
- ### AI编程
  - 学习行为树、状态机等游戏AI技术。
  - 探索机器学习在游戏NPC、关卡生成等方面的应用。

## 二、项目经验
- ### 独立游戏开发
  - 构思并完成至少一款完整的独立游戏。
  - 参与 Game Jam 等极限开发挑战。
- ### 开源项目贡献
  - 为知名的开源游戏引擎或工具贡献代码。
  - 参与开源社区，与其他开发者协作。

## 三、个人品牌
- ### 技术博客
  - 撰写博客，分享学习心得和项目经验。
- ### 作品集
  - 持续更新和打磨个人作品集。
- ### 社区分享
  - 在技术论坛或线下活动中进行分享。
