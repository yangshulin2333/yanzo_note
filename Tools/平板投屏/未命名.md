# 小米平板无线投屏教程 (Scrcpy)

## 1. 工具下载
- **工具名称**: Scrcpy
- **GitHub 地址**: [https://github.com/Genymobile/scrcpy](https://github.com/Genymobile/scrcpy)
- **下载指南**: 
  - 进入 Releases 页面。
  - 下载 `scrcpy-win64-vX.X.zip` (请下载 Assets 下的压缩包，不要下载 Source code)。

## 2. 首次准备工作 (仅需做一次)
- **开启开发者模式**: 
  - `设置` -> `关于平板` -> 连续点击 7 次 **“OS 版本号”** (HyperOS)。
- **开启 USB 调试**: 
  - `设置` -> `更多设置` -> `开发者选项` -> 开启 **“USB 调试”**。

## 3. 开启无线端口 (每次重启平板后需操作)
> **注意**：这一步必须通过 USB 线连接进行授权。

1. **插线**: 用数据线连接平板和电脑。
2. **选模式**: 平板下拉通知栏，将 USB 连接模式从“仅充电”改为 **“文件传输 / MTP”**。
3. **输命令**: 
   - 在电脑 Scrcpy 文件夹内双击 `open_a_terminal_here.bat`。
   - 输入命令并回车：
     ```bash
     adb tcpip 5555
     ```
4. **拔线**: 当看到提示 `restarting in TCP mode port: 5555` 后，即可拔掉数据线。

## 4. 日常连接步骤 (纯无线)
1. **查看 IP**: 
   - 平板 `设置` -> `WLAN` -> 点击已连 WiFi -> 查看 **IP 地址** (例如 `192.168.0.3`)。
2. **连接设备**: 
   - 在电脑黑窗口 (终端) 输入：
     ```bash
     adb connect 192.168.0.3
     ```
   - *(请将 `192.168.0.3` 替换为您平板实际的 IP)*
3. **启动投屏**: 
   - 双击文件夹里的 `scrcpy.exe`。

## 5. 避坑指南 (Troubleshooting)
- **连接超时 / 失败**: 
  - 检查电脑是否开启了 **代理软件** (如 Sing-box / V2Ray)。
  - **解决方法**: 暂时关闭代理软件 -> 执行 `adb connect` -> 连接成功后再开启代理。
- **找不到设备 (No device found)**: 
  - 检查 USB 连接模式是否误选了“仅充电”，必须是 **“文件传输”** 才能被 ADB 识别。
