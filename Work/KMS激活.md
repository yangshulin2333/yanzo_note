从你提供的截图来看，你下载的是一个 KMS 激活服务的源码/静态网页项目，而**激活 Windows 或 Office 并不需要运行这些代码文件**。

你只需要以**管理员身份**运行“命令提示符”（CMD），并依次输入截图左侧 README 中提到的三条命令即可。具体操作步骤如下：

## 第一步：打开管理员终端

1. 按下键盘上的 **Win + S** 键，输入 `cmd`。
    
2. 在搜索结果中，右键点击“命令提示符”，选择“**以管理员身份运行**”（这一步非常关键）。
    

---

## 第二步：依次执行激活命令

#### 1. 设置激活服务器

将你的激活服务器指向截图里提到的地址。在黑窗口中粘贴并回车：

DOS

```
slmgr -skms skms.netnr.eu.org
```

_注：如果这个服务器连接失败，也可以尝试其他的公共 KMS 地址。_

#### 2. 安装密钥 (GVLK)

你需要根据你当前的系统版本（如专业版、企业版）安装对应的 GVLK 密钥。

DOS

```
slmgr -ipk [你的版本对应密钥]
```

> **💡 贴心提醒：** 如果你不确定密钥是多少，通常 Windows 专业版常用的密钥是：`W269N-WFGWX-YVC9B-4J6C9-T83GX`。
> 
> 对应的完整命令即为：
> 
> `slmgr -ipk W269N-WFGWX-YVC9B-4J6C9-T83GX`

#### 3. 激活系统

最后一步，让系统向服务器发送激活请求：

DOS

```
slmgr -ato
```

---

## 常见问题排查

- **如何查看激活状态？**
    
    执行完后，你可以输入 `slmgr -xpr` 查看是否显示“计算机已永久激活”或“批量激活将于某日过期”（KMS 激活通常有效期为 180 天，到期会自动续期）。
