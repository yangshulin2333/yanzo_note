## PART 1: THE DEEP DIVE (WiFi Architecture)

### Technical Specifications

- **Spectral Analysis:** Features a dedicated radio for continuous environmental scanning to identify interference.
    
- **PRISM™ Technology:** Active RF filtering that blocks adjacent channel 5 GHz interference to improve SNR and stability.
    
- **WiFi 7 Features:** * **MLO (Multi-Link Operation):** Combines multiple bands simultaneously to reduce latency and maximize throughput.
    
    - **Puncturing:** Allows higher-width channels to notch out noise and operate non-continuously.
        
- **Management & Security:**
    
    - **Coverage Visualization:** Real-time channel distribution mapping within UniFi Site Manager.
        
    - **Flexible Authentication:** Supports UniFi Identity, local/AD/Cloud RADIUS, and Gateway-hosted RADIUS.

## PART 5: THE DEEP DIVE (Cloud Gateway Platforms)

### AI-Enhanced Cybersecurity Features

- **IDS/IPS:** Real-time threat database used to detect and block suspicious internal and external network activity.
    
- **Zone-Based Firewall:** Simplifies security by grouping networks (LAN, WAN, Guest, IoT) into zones for scalable policy management.
    
- **Site Magic SD-WAN:** License-free, one-click setup for Site-to-Site VPN and SD-WAN through UniFi Site Manager.
    
- **Shadow Mode (High Availability):** Utilizes VRRP to link a secondary gateway that instantly takes over if the primary fails.
